Мои навыки

Без чего невозможно создать современный сайт

html

Стандартизированный язык разметки документов для просмотра веб-страниц в браузере.

css3

Каскадные таблицы стилей — формальный язык описания внешнего вида веб-страницы, написанного с использованием языка разметки HTML.

JavaScript

Мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили.

Bootstrap

Свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления.

JQuery

Набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML.

React

JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов.

Sass

Предназначен для увеличения уровня абстракции CSS-кода и упрощения файлов каскадных таблиц стилей.

Git

Распределенная система контроля версий и управления исходным кодом с упором на скорость.

Figma

Разработка интерфейсов и прототипирования с возможностью организации совместной работы в режиме реального времени.
